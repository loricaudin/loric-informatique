Résoudre l'erreur : 0x80070780

(0x80070780) : Le système ne peut pas accéder au fichier.

Version 1.0 (créé le 12/09/2025, modifié le 12/09/2025)

Fonctionne sur : Windows 10 et supérieur

Vous avez des fichiers OneDrive et vous tentez de modifier un fichier, de le déplacer ou de renommer et vous n'y arrivez plus à cause de cette erreur. Pas de panique, nous avons peut-être la solution.

Ce que vous avez besoin :

  • Juste une page web OneDrive
  • Dans le pire des cas, le fichier d'installation de OneDrive

Avant de commencer, assurez-vous que vos fichiers peuvent être ouverts sur OneDrive en ligne. Si c'est le cas, alors votre application OneDrive s'est peut-être plantée (peut-être dû à une mise à jour Windows ou OneDrive qui n'a pas marché ou un retour en arrière).

Fermer ensuite votre navigateur, ouvrir l'explorateur de fichiers et supprimer 1 fichier ayant ce problème.

Attendre un petit instant, et bizarrement, le fichier revient tout seul, vérifier que vous pouvez de nouveau le modifier et le renommer

Si le problème est résolu pour ce fichier, supprimer tous les autres fichiers ayant ce problème

Si le problème n'est pas réglé ou que le fichier n'est pas réapparu après 5 minutes, alors votre installation OneDrive ne marche plus.
