Résoudre l'erreur : 0x80070422

(0x80070422) : Impossible de mettre à jour les définitions de virus et de logiciels espions.

Version 1.1 (créé le 05/12/2019, modifié le 12/09/2025)

Fonctionne sur : Windows 8.1 et 10

Cette erreur est moins connue chez les nouveaux Windows, mais si vous possédez cette erreur sur Windows 10 version 1607 ou inférieur, Windows 8.1, 7 ou Vista, redirigez-vous sur cette page.

Si vous avez installé Wub, désactivez le, puis réactivez le après la mise à jour.
