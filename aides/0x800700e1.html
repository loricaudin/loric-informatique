Résoudre l'erreur : 0x800700e1

(0x800700E1) : Impossible de terminer l'opération, car le fichier contient un virus ou un logiciel potentiellement indésirable.

Version 1.1 (créé le 02/08/2022, modifié le 03/09/2024)

Fonctionne sur : Windows 10 et supérieur

Cette erreur vous indique que Windows Defender (ou Microsoft Defender) a trouvé un virus sur ce logiciel. Cela l'empêche de faire toute manipulation sur ce fichier.

Que faire ?

Attendez une minute que le fichier se supprime tout seul, par mesure de sécurité, déconnectez le PC d'Internet, si le problème n'est pas résolu, désactivez votre antivirus et supprimer immédiatement le fichier (en faisant Maj + Suppr)
