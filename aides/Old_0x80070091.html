Résoudre l'erreur : 0x80070091

(0x80070091) : Le répertoire n’est pas vide :

12/07/2020

Cette erreur est assez connu lorsque vous avez beaucoup de fichiers et de dossiers à supprimer, mais cela affiche cet erreur. Malheureusement, nous n’avons pas d’autres choix que de supprimer les fichiers d’abord, puis les dossiers un par un.

Si le problème n’est pas résolu, vérifier que les fichiers cache sont supprimés dans l’onglet Affichage, puis cochez Eléments masqués

Si des fichiers cache apparaissent, supprimez-les tous et vérifiez que le problème est résolu.
