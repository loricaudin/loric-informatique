I.1. Syntaxe

}

}

I.2. Où écrire le code CSS ?

Soit dans un fichier (ex : style.css), mais entre les balises ‹head›‹/head›, il faut rajouter : ‹link rel="stylesheet" href="style.css"/›

Soit entre les balises ‹style›‹/style›, placées dans ‹body›‹/body› (à éviter)

II. Les différents sélecteurs

II.1. Sélecteurs simples

Sélecteur Description
* Sélectionner toutes les balises
balise1 Sélectionner toutes les balises balise1
balise1, balise2 Sélectionner toutes les balises balise1 et balise2
balise1 › balise2 Sélectionner toutes les balises balise2 fils directs de balise1
balise1 balise2 Sélectionner toutes les balises balise2 qui sont dans balise1
balise1 + balise2 Sélectionner la première balise balise2 qui suit la balise balise1
balise1 ~ balise2 Sélectionner toutes les balises balise2 qui sont après balise1 et au même niveau
balise1[attribut1] Sélectionner toutes les balises balise1 qui ont un attribut attribut1
balise1[attribut1="valeur1"] Sélectionner toutes les balises balise1 qui ont un attribut attribut1 avec pour valeur valeur1
balise1.classe1 Sélectionner les balises balise1 qui ont pour classe classe1
.classe1 Sélectionner toutes les balises qui ont pour classe classe1
balise1#id1 Sélectionner une balise balise1 qui a pour identifiant id1
#id1 Sélectionner une balise qui a pour identifiant id1

Propriété Résultat Description
color: couleur; Mon texte Changer la couleur du texte
background-color: couleur; Mon texte Changer l'arrière-plan

III.XXX. Propriétés sur le texte

Propriété Valeurs possibles Description
text-decoration: valeur;
  • none
  • underline
 Changer la décoration du texte (à utiliser pour retirer la décoration d'un lien si besoin)
text-transform: valeur;
  • none
  • capitalize
  • uppercase
  • lowercase
 Transformer le texte
text-align: valeur;
  • left
  • right
  • center
  • justify
 Changer l'alignement du texte

III.XXX. Propriétés sur la police

Propriété Valeurs possibles Description
font-style: valeur;
  • normal
  • italic
  • oblique
 Changer le style de la police
font-weight: valeur;
  • normal
  • bold
  • bolder
 Changer l'épaisseur de la police (pour mettre en gras)
font-variant: valeur;
  • normal
  • small-caps
 Changer l'apparence de la police (small-caps met tout en capitale avec les lettres en majuscule plus grandes)
font-family: valeur1, valeur2…;
  • Calibri
  • Arial
  • serif
  • sans-serif
    • …
Remplacer la police par la première disponible de la liste (si la première n'existe pas, c'est la deuxième qui est chargée…)
font-size: taille;
  • 20px
  • 20pt
  • 0.2in
  • 0.5cm
  • 5mm
  • 2pc
 Changer la taille du texte
