Table des matières :

Prise en main

Créer un projet

Avant de créer un projet, vous devez installer NodeJS. Pendant le processus d'initialisation, Vite sera installé automatiquement.

Exécuter la commande : npm create vite@latest ./ (si des packages doivent être installés, entrez y)

Project name : MonProjet

Package name : monprojet

Framework : React

Variant : JavaScript

Entrer ensuite dans le dossier MonProjet et entrer la commande : npm install

Développer le projet

Dans le dossier de votre projet, entrez la commande : npm run dev (il est conseillé de l'exécuter dans le terminal de Visual Studio Code s'il est installé)

Pour arrêter le serveur, faites Ctrl + C dans la console et entrez o s'il est demandé.
