HTML

Version 0.3 (créé le 20/04/2024, modifié le 20/04/2024)

HTML (HyperText Markup Language) est le langage utilisé pour structurer une page web et son contenu. On peut par exemple organiser le contenu en un ensemble de paragraphes, une liste d'éléments, utiliser des images ou des tableaux de données.

Outils nécessaires :

  • Navigateur Internet à jour (ex : Firefox ou Chrome)
  • Un logiciel de codage (Visual Studio Code (recommandé) ou Notepad++)
  • Validateur HTML : https://validator.w3.org/

Table des matières :

Bases

Syntaxe

<!DOCTYPE html>
<html lang="fr-FR">
    <head>
        <title>Titre</title>
        <meta charset="utf-8"/>
        <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
        <link rel="icon" href="image/icone.png"/>
    </head>
    <body>
        <balise1>contenu1</balise1>
        <balise2>contenu2</balise2>
        <balise3>contenu3</balise3>
    </body>
</html>

Où écrire le code HTML ?

Dans un fichier .html ou dans un fichier .php pour les serveurs compatibles, entre les balises <body></body>

Le head

Le head permet la configuration de la page web.

Balise Description
<title>Mon titre</title> Changer le titre au niveau de l'onglet
<meta charset="utf-8"/> Changer l'encodage
<meta name="viewport" content="width=device-width, initial-scale=1.0"/> Adapter l'affichage pour les appareils mobiles
<meta name="robots" content="noindex, nofollow"/> Ne pas indexer cette page par les moteurs de recherches (si le site est disponible en ligne)
<link rel="icon" href="icone.png"/> Changer l'icône au niveau de l'onglet
<link rel="stylesheet" href ="style.css"/> Appliquer une feuille de style à la page HTML (un attribut media peut être renseigné)
Certaines balises sont auto-fermantes comme <link/>. En HTML, le "/" n'est pas obligatoire, mais conseillé pour une meilleure lisibilité. Dans d'autres langages, il peut être obligatoire (notamment en XML et dans les frameworks JS)

Astuce : Sous certains outils de développement comme Visual Studio Code, vous pouvez utiliser Emet afin de générer automatiquement l'entête du fichier HTML. Il suffit d'entrer "!" et la structure de base se génère toute seule.

Les liens

Syntaxe : <a href="lien">Cliquez ici</a>

Attribut Description
href="lien" Rediriger la page vers une autre page au clic sur le lien (chemin absolu ou relatif)
href="#id" Rediriger la page vers une zone de la page HTML
target="_self" Ouvrir la page dans le même onglet (option par défaut)
target="_blank" Ouvrir la page dans un nouvel onglet
target="_parent" Ouvrir la page dans la fenêtre principale
target="_top" Ouvrir la page dans une fenêtre séparée

Les images

Syntaxe : <img src="lien" alt="texte-de-remplacement"/>

Attribut Description
src="lien" Récupérer la source de l'image (chemin absolu ou relatif)
alt="texte-de-remplacement" Remplacer l'image par du texte si le chargement a échoué

Les formulaires

Un formulaire permet l'envoi de données vers un serveur, avec pour action le lien vers une page PHP (ne fonctionne pas avec pour action un fichier .html).

Créer un formulaire

Syntaxe : <form method="methode" action="lien">champs</form>

Méthode Description
GET Écrire les données saisies dans l'URL
POST Envoyer les données dans une requête HTTP (sans écrire dans l'URL)
