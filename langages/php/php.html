Les variables

Instruction Description
let variable = valeur; Déclare une nouvelle variable dont la portée est limitée à celle du bloc dans lequel elle est déclarée
var variable = valeur; Déclare une nouvelle variable globale ou locale à une fonction (sans distinction des blocs utilisés dans la fonction)
var a; Déclare la variable a sans affecter de valeur (contient null)
a = 5; Affecte 5 à une variable (si la variable n'est pas déclarée (ce qui est déconseillé de faire), alors elle est considérée comme globale, mais accessible même en dehors d'une fonction)
a += 1; Équivalent à a = a + 1;
a -= 1; Équivalent à a = a - 1;
b = a++; Équivalent à b = a; a = a + 1;
b = ++a; Équivalent à a = a + 1; b = a;
b = a--; Équivalent à b = a; a = a - 1;
b = --a; Équivalent à a = a - 1; b = a;
const pi = 3.14; Crée une variable constante (non modifiable)

Les tableaux

Instruction Description
var tableau = []; Crée un tableau vide

 Renvoie la valeur du tableau à la position i (l'indice de la première valeur est 0)

Conditions

Les conditions renvoient true si elle est respectée et false sinon

Opérateurs de comparaison

Condition Description de ce que vérifie la condition
a == b a égal à b (seulement en contenu : 1="1")
a === b a égal à b (en type et en contenu : 1≠"1")
a < b a strictement inférieur à b
a > b a strictement supérieur à b
a <= b a inférieur ou égal à b
a >= b a supérieur ou égal à b
a != b a n’est pas égal à b (seulement en contenu : 1="1")
a !== b a n’est pas égal à b (en type et en contenu : 1≠"1")
|| À mettre entre deux conditions, permet d’avoir une des deux conditions qui doit être vraie
&& À mettre entre deux conditions, permet d’avoir deux conditions qui doivent être vraie
!condition La condition doit être fausse

Tests de conditions

Instruction Description
if (condition1) {
    instruction1;
} 		Si condition1 est vraie, alors on exécute instruction1
if (condition1) {
    instruction1;
} else {
    instruction2;
} 		Si condition1 est vraie, alors on exécute instruction1, sinon, on exécute instruction2
if (condition1) {
    instruction1;
} else if(condition2) {
    instruction2;
} else {
    instruction3;
} 		Si condition1 est vraie, alors on exécute instruction1, sinon, si condition2 est vraie, on exécute instruction2, sinon, on exécute instruction3
switch (nombre) {
    case a:
        instruction1;
        break;
    case b:
    case c:
        instruction2;
        break;
    default:
        instruction3;
} 		Si nombre == a, alors on exécute instruction1, sinon, si nombre == b ou c, on exécute instruction2, sinon, on exécute instruction3
a = (condition1) ? 1 : 0; Si condition1 est vraie, a prend la valeur 1, sinon 0.
try {
    instruction1;
} catch(erreur) {
    instruction2;
} 		Si instruction1 provoque une erreur, on exécute instruction2 (si throw message; est placé dans instruction1, alors instruction2 est exécuté avec erreur = message)

Boucles

Instruction Description
for (let i = d; i < f; i++) {
    instruction1;
} 		On répète f-d fois instruction1 pour i allant de d compris à f non compris
for (let i = d; i < f; i+=p) {
    instruction1;
} 		On répète (f-d)/p fois instruction1 pour i allant de d compris à f non compris avec pour pas égal à p
for (let i in tableau) {
    instruction1
} 		On parcours le tableau (ou une chaîne de caractères) pour i allant de 0 à la longueur de tableau
for (let elt of tableau) {
    instruction1
} 		On parcours le tableau (ou une chaîne de caractères) pour elt prenant toutes les valeurs de tableau
while (condition) {
    instruction1;
} 		On répète jusqu’à ce que condition soit fausse (peut ne pas être répété)
do {
    instruction1;
} while(condition); 		On répète jusqu’à ce que condition soit fausse (est forcément répété une fois)
break; Permet de sortir d’une boucle sans la terminer (fortement déconseillé)

Les fonctions et les classes

Les fonctions

Créer une fonction

function maFonction(variable1, variable2…) {
    instructions;
}

Retourner une valeur de la fonction

return variable;

Faire un appel à la fonction

variable = maFonction(valeur1, valeur2…);

ou (s'il n'y a pas de variable de retour)

maFonction(valeur1, valeur2…);

Si une valeur n'a pas de valeur renseignée, alors elle prend comme valeur undefined

Les classes

Créer une classe

class MaClasse {

    static variables_statiques = valeurs;

    constructor(mVariable1, mVariable2…) {
        this.variable1 = mVariable1;
        this.variable2 = mVariable2;
    }
}

Définir la classe dans un objet

var monObjet = new MaClasse(valeur1, valeur2…);

Créer une méthode (dans une classe)

get maMethode(variable1, variable2…) {
    instructions;
}

Vous pouvez également créer des méthodes statiques en remplaçant get par static.

Faire un appel d’une méthode autre que le constructeur ou d’une variable dans une classe

this.maMethode(valeur1, valeur2…)

this.variable1 = valeur1;

Faire un appel d’une méthode autre que le constructeur ou d’une variable en dehors d'une classe

monObjet.maMethode(valeur1, valeur2…)

monObjet.variable1 = valeur1;

Il est préférable d'utiliser des méthodes (appelées getter et setter) pour modifier les variables non statiques.

Faire un appel d’une méthode ou d’une variable statique en dehors d'une classe

MaClasse.maMethode(valeur1, valeur2…)

MaClasse.variable1 = valeur1;
